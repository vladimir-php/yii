Описание настроек локальной машины
Apache/2.2.4
MySQL/5.0.45 
PHP/5.2.4

файлы проекта находятся в откытом репозитории https://github.com/vladimir-php/yii/
их нужно скопировать в папку /home с требуемым названием сайта для локальной версии

При локальной установке нужно испортировать файл БД - /protected/data/schema.mysql.sql в уже созданную пустую базу данных с названием yiitest
(если название не подходит можно взять другое и изменить в конфиге)
